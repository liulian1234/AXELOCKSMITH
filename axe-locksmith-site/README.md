# AXE Locksmith Website

This is a static landing page for GitHub Pages.

## How to use

1. Replace all `+6500000000` phone numbers in `index.html` with your real Singapore phone number.
2. Replace the WhatsApp number inside all `https://wa.me/6500000000` links.
3. Upload all files to your GitHub repository.
4. In GitHub: Settings → Pages → Deploy from branch → main → root.
5. Connect Google Ads to the published GitHub Pages URL.

## Files

- `index.html` — page content
- `style.css` — design and responsive layout
- `script.js` — mobile menu and dropdown behaviour
- `assets/logo.svg` — editable SVG logo
- `assets/hero-locksmith.svg` — hero illustration

## Suggested edits before advertising

- Add your real company registration details if you want to build trust.
- Add real photos of your tools/work after you have them.
- Add exact service areas you can reach within your planned timing.
- Avoid claiming a guaranteed arrival time unless you can really fulfil it.
